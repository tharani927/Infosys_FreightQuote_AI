# Milestone 1

## User Authentication Module

This milestone implements a secure user authentication system using Streamlit.

### Features
- User Signup
- User Login
- Forgot Password using Security Question
- Forgot Password using Gmail OTP
- JWT Authentication
- User Dashboard
- Admin Dashboard

### Technologies Used
- Python
- Streamlit
- SQLite
- JWT
- bcrypt
- Gmail SMTP
- ngrok

### How to Run
1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Configure Colab Secrets.
4. Run the notebook.
5. Open the ngrok URL.
